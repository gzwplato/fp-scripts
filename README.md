# fp-scripts
Scripts to compile FreePascal, Lazarus and MSE on Windows

To use these scripts without modifications, you should use this structure of directories:

    freepascal
        |- binutils
            |- 3.0.0
            |- 3.0.1
            |- 3.1.1
            |- laz        
        |- compilers
            |- 3.0.0
            |- 3.0.1
            |- 3.1.1
            |- laz        
        |- lazarus
            |- 1.5
            |- 1.5-config
            |- 1.7
            |- 1.7-config
        |- lib
        |- mse
        |- scripts
